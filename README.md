### Prerequisites



### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/MishTails/Capstone-ESPNBasketball
   ```
2. Install NPM packages on the front end
   ```sh
   npm install
   ```
3. Install NPM packages on the back end
   ```sh
   pipenv install
   ```
